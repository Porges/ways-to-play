# <span lang="egy">𓅓𓐱𓎛𓐰𓈖𓏌</span> · <span lang="egy-Latn" class="noun">Mehen</span>

<span lang="egy-Latn" class="aka noun">Mehen</span> (<span lang="egy" class="aka">𓅓𓐱𓎛𓐰𓈖𓏌</span>) is an ancient Egyptian board game that was played on a distinctive spiral board shaped like a coiled serpent.

The game appears in art up until the Middle Kingdom (2055–1650 <abbr>BCE</abbr>). It much later re-appeared in the Saite period (26th Dynasty, 664–525 <abbr>BCE</abbr>), as part of a general revival of Old Kingdom artistic styles.[@SerpentBoard 89] However, these later depictions are likely only decorative — the game itself was probably no longer played at this point.
